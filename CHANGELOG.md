## [Web v0.1.0] - 2026-01-07

### Added

- Initial project setup using a Hybrid Stack: [Elysia](https://elysia.dev/) (Bun) for Core API and [Axum](https://github.com/tokio-rs/axum) (Rust) for Performance services.
- Frontend: [NextJS](https://nextjs.org/) for Web and [Tauri v2](https://tauri.app/) for Cross-platform.
- Database: [PostgreSQL](https://www.postgresql.org/) (Replacing MongoDB).
- AI & Games: Rust-based AI and Unity WebGL integration.
- Added `README.md` and `CHANGELOG.md` files.
- Added `docker-compose.yaml` file for backend.
- Create cross-platform with [Tauri2](https://tauri.app/).
